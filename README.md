Docker ImapSync
==================================================

1. Change env variabiles in docker-compose.yml with your credentials
```
    - HOST1=
    - USER1=
    - PASSWORD1=
    - OTHER1=           (for other arguments)
    - HOST2=
    - USER2=
    - PASSWORD2=
    - OTHER2=		(for other arguments es: --ssl2 --port=993)

```

2. docker-compose up 


The first run take a few minute, because docker image must build.
